# 🌍 EcoSphere – ESG Management Platform

> An intelligent ESG (Environmental, Social, Governance) Management Platform that helps organizations monitor sustainability, track carbon emissions, improve employee engagement, ensure governance compliance, and drive ESG initiatives through analytics, gamification, and AI.

---

## 🚀 Overview

EcoSphere is a full-stack web application developed as a hackathon project to simplify ESG management for organizations.

The platform enables companies to:

- 🌱 Track Carbon Emissions
- 📊 Monitor Environmental Performance
- 👥 Manage CSR Activities
- 🏆 Gamify Employee Participation
- 📜 Ensure Governance Compliance
- 🤖 Generate AI-powered Sustainability Insights
- 📈 Visualize ESG Reports & Analytics

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/mradanshi/EcoSphere_ESG_Management_Platform-Team_Nexora.git
```

Move into the project

```bash
cd EcoSphere_ESG_Management_Platform-Team_Nexora
```

Backend

```bash
cd backend
```

Run Spring Boot

```bash
mvn spring-boot:run
```

Frontend

```bash
cd frontend
npm install
npm run dev
```
